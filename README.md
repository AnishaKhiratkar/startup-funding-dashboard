# Indian Startup Funding Analysis Dashboard

This project is an interactive data analytics dashboard built using Streamlit to explore and analyze Indian startup funding data.  
It provides insights from three perspectives: company-level analysis, investor behavior analysis, and overall ecosystem trends.

---

## Project Overview

The goal of this project is to understand funding patterns in the Indian startup ecosystem using real-world data.  
The dashboard enables users to explore funding trends, investor preferences, sector dominance, and city-wise investment distribution through interactive visualizations.

---

## Dashboard Structure

### 1. Company Perspective
- Startup-wise funding history
- Funding rounds and investors
- Sector, sub-sector, and city details
- Year-wise funding timeline

### 2. Investor Perspective
- Most recent investments
- Biggest investments by amount
- Sector, stage, and city investment preferences
- Year-on-year investment trend
- Similar investors based on common startups

### 3. General Analysis
- Key performance indicators:
  - Total funding amount
  - Maximum single funding
  - Average ticket size
  - Number of funded startups
- Month-on-month funding trends (amount and deal count)
- Sector-wise funding analysis (count and total amount)
- City-wise funding distribution
- Top funded startups
- Top investors by deal count

## How to Run the Project

1. Clone the repository
   https://github.com/AnishaKhiratkar/startup-funding-dashboard.git

2.Navigate to the project folder

   cd startup-funding-dashboard
 
3.Install dependencies

  pip install -r requirements.txt

4.Run the application
 
  streamlit run app.py

5.Open the local URL shown in the terminal in your browser.
